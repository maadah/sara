# تحسينات الشات بوت - نظام الإشعارات والرد المباشر على استعلامات الطلبات

## التاريخ: 2025-12-18

## الميزات الجديدة المضافة

### 1. فحص مباشر لحالة الطلب من قاعدة البيانات ✅
**الدالة:** `checkOrderStatus()`

عندما يسأل العميل عن طلبه، النظام الآن:
- يفحص قاعدة البيانات **مباشرة** بدون الذهاب للـ AI
- يجلب آخر طلب للعميل من جدول `online_orders`
- يعطي رد دقيق بناءً على **الحالة الفعلية** للطلب

#### الكلمات المفتاحية المدعومة:
- "وين طلبي"
- "يمته يوصل"
- "متى يوصل"
- "كم وصل"
- "شنو تفاصيل طلبي"
- "اريد طلبي الحالي"
- "حالة طلبي"

#### أمثلة الردود حسب حالة الطلب:

**Pending/Confirmed:**
```
طلبك #ORD-001 مؤكد وقيد التجهيز 📦
راح يوصلك خلال 1-3 أيام إن شاء الله

المنتجات: قميص × 1
```

**Processing:**
```
طلبك #ORD-001 قيد التجهيز حالياً 👨‍💼
راح يشحن قريباً جداً وراح يوصلك خلال 1-2 يوم

المنتجات: قميص × 1
```

**Shipped:**
```
طلبك #ORD-001 بالطريق إليك! 🚚
راح يوصلك خلال 24 ساعة إن شاء الله

المنتجات: قميص × 1
```

**تفاصيل كاملة (عند السؤال عن التفاصيل):**
```
📋 تفاصيل طلبك:

🔢 رقم الطلب: #ORD-001
📦 المنتجات: قميص × 1
💰 المبلغ الكلي: 15,000 IQD
📊 الحالة: قيد التجهيز
📅 تاريخ الطلب: 2025-12-17

⏰ التوصيل المتوقع: خلال 1-2 يوم
```

### 2. كشف الأسئلة التي تحتاج رد بشري 🔔
**الدالة:** `requiresImmediateHumanResponse()`

النظام يكتشف تلقائياً الأسئلة والمشاكل المعقدة التي تحتاج **تدخل بشري**:

#### أ) الشكاوى والمشاكل:
- شكوى
- مشكلة
- غلط / خطأ
- ما وصلني / ما جاني
- متأخر
- تالف / مكسور
- ناقص
- اريد استرجاع / ارجاع / استبدال

#### ب) الطلبات الخاصة:
- تخصيص
- تصميم خاص
- طلب خاص
- طباعة اسمي
- اضافة صورة
- ممكن تعدل

#### ج) أسئلة الدفع:
- دفع فيزا
- بطاقة ائتمان
- زين كاش
- كي كارد
- تقسيط / دفعات

#### د) طلبات الجملة:
- جملة / بالجملة
- أعداد كبيرة (50+ قطعة)

### 3. ردود انتظار ذكية أثناء تدخل الموظف ⏳
**الدالة:** `getWaitingMessage()`

عندما يحتاج السؤال لرد بشري، يرسل النظام رد **مهذب** للعميل حسب نوع السؤال:

#### أسئلة حالة الطلب:
```
لحظات أتأكد من حالة الطلب مالتك... 📦
```

#### أسئلة التوفر:
```
ثواني أتأكد إذا متوفر... 🔍
```

#### أسئلة السعر:
```
لحظة أشوف السعر الكلي... 💰
```

#### أسئلة الاسترجاع:
```
لحظات أتأكد من سياسة الاسترجاع... 📋
```

#### ردود عامة (عشوائية):
- "لحظات، دعني أتأكد وأرد عليك... ⏳"
- "ثواني أتحقق من المعلومات... 🔍"
- "دقيقة واحدة أتأكد... ⌛"
- "لحظة، خليني أشوف... 👀"

---

### 4. نظام الإشعارات المحسّن 🔔
**الدالة:** `notifyHumanAgent()`

عندما يحتاج سؤال لرد بشري:

#### يتم إنشاء إشعار يحتوي على:
- نوع الإشعار: `human_agent_needed`
- العنوان: "تحتاج محادثة إلى تدخل بشري"
- الرسالة: "المحادثة مع [اسم العميل] تحتاج إلى موظف"
- البيانات المرفقة:
  - `conversation_id`: معرف المحادثة
  - `lead_id`: معرف العميل
  - `customer_message`: رسالة العميل الأصلية

#### ميزة الإشعار:
- **قابل للنقر**: عند النقر على الإشعار، ينتقل الأدمن مباشرة للمحادثة
- **يحتوي على السياق**: يعرض رسالة العميل الأخيرة

---

## تسلسل العمل (Workflow)

```
1. عميل يرسل رسالة
   ↓
2. هل هو سؤال عن حالة طلب؟
   ├─ نعم → فحص مباشر من قاعدة البيانات → رد فوري
   └─ لا → التالي
   ↓
3. هل يحتاج رد بشري (شكوى، طلب خاص، إلخ)؟
   ├─ نعم → إشعار للأدمن + رد انتظار للعميل
   └─ لا → معالجة عبر AI عادي
   ↓
4. إذا AI قرر يحتاج رد بشري
   → إشعار للأدمن + رد انتظار
```

## أمثلة على السيناريوهات

### السيناريو 1: سؤال عن حالة الطلب ✅
**العميل:** "يمته يوصل طلبي؟"
**النظام:** 
1. يكتشف أنه سؤال عن الطلب
2. يفحص قاعدة البيانات مباشرة
3. يرد حسب الحالة الفعلية: "طلبك #ORD-001 قيد التجهيز... راح يوصلك خلال 1-2 يوم"
**لا يذهب للـ AI - رد فوري ودقيق!**

### السيناريو 2: شكوى من العميل 🔔
**العميل:** "الطلب وصلني تالف!"
**النظام:**
1. يكتشف كلمة "تالف" (شكوى)
2. يرسل إشعار للأدمن
3. يرد للعميل: "لحظات أتأكد من سياسة الاسترجاع... 📋"
4. الأدمن يستلم إشعار: "المحادثة مع زيد تحتاج إلى موظف. آخر رسالة: الطلب وصلني تالف!"
5. الأدمن ينقر على الإشعار → ينتقل للمحادثة → يرد يدوياً

### السيناريو 3: طلب خاص 🔔
**العميل:** "ممكن تطبع اسمي على التيشرت؟"
**النظام:**
1. يكتشف "طبع اسمي" (طلب خاص)
2. يرسل إشعار للأدمن
3. يرد للعميل: "ثواني أتحقق من المعلومات... 🔍"

### السيناريو 4: سؤال عادي (AI) 💬
**العميل:** "عندكم قمصان؟"
**النظام:**
1. ليس سؤال عن طلب
2. ليس شكوى أو طلب خاص
3. يذهب للـ AI العادي
4. AI يرد: "نعم عندنا قمصان! السعر 15,000 دينار 👔"

## التكامل مع الواجهة الأمامية

### لعرض الإشعارات في لوحة التحكم:

```javascript
// جلب الإشعارات
axios.get('/api/notifications')
  .then(response => {
    response.data.forEach(notification => {
      if (notification.type === 'human_agent_needed') {
        // عرض إشعار
        showNotification({
          title: notification.title,
          message: notification.message,
          onClick: () => {
            // الانتقال للمحادثة
            window.location.href = `/conversations/${notification.data.conversation_id}`;
          }
        });
      }
    });
  });
```

## المزايا الرئيسية

### ✅ دقة عالية
- فحص مباشر من قاعدة البيانات
- لا اعتماد على AI للأسئلة البسيطة

### ⚡ سرعة الرد
- ردود فورية لأسئلة حالة الطلب
- لا انتظار لـ AI API

### 👨‍💼 تجربة أفضل للموظف
- إشعارات واضحة
- سياق كامل للمشكلة
- انتقال مباشر للمحادثة

### 😊 تجربة أفضل للعميل
- ردود دقيقة عن طلبه
- رسائل انتظار مهذبة
- شعور بالاهتمام
