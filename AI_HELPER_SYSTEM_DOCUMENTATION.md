# AI-Helper Dashboard System Documentation
## نظام لوحة تحكم مساعد الذكاء الاصطناعي

---

## 📋 نظرة عامة

نظام **AI-Helper (AI-H)** هو نظام متكامل يساعد المتجر على تحسين أداء الشات بوت من خلال:
1. **قاعدة المعرفة** (Knowledge Base) - تعليم AI إجابات جديدة
2. **الردود السريعة** (Fast Replies) - إدارة رسائل الترحيب والردود التلقائية
3. **الأسئلة غير المُجابة** (Unanswered Questions) - تتبع ما لا يعرف AI الإجابة عليه
4. **نظام الإشعارات** - تنبيه الأدمن بالأسئلة المعلقة

---

## 🗄️ قاعدة البيانات

### 1. جدول `ai_knowledge_base`

يخزن أزواج الأسئلة والأجوبة التي يتعلم منها AI.

#### الحقول:
| الحقل | النوع | الوصف |
|------|------|------|
| `id` | bigint | المعرف الفريد |
| `user_id` | bigint | معرف المتجر |
| `question` | text | السؤال |
| `answer` | text | الإجابة |
| `category` | string | التصنيف (توصيل، منتجات، دفع) |
| `keywords` | json | كلمات مفتاحية مستخرجة |
| `usage_count` | int | عدد مرات الاستخدام |
| `status` | enum | active, inactive, draft |
| `is_verified` | boolean | هل تم التحقق من قبل الأدمن |
| `use_for_training` | boolean | استخدام في تدريب AI |
| `priority` | int | الأولوية (أعلى = أهم) |

#### المميزات:
- ✅ **Full-Text Search** للبحث السريع
- ✅ **Soft Deletes** للحفاظ على البيانات
- ✅ **استخراج تلقائي للكلمات المفتاحية**
- ✅ **تتبع الاستخدام**

---

### 2. جدول `ai_fast_replies`

يخزن الردود السريعة والرسائل التلقائية.

#### الحقول:
| الحقل | النوع | الوصف |
|------|------|------|
| `id` | bigint | المعرف الفريد |
| `user_id` | bigint | معرف المتجر |
| `name` | string | اسم الرد (مثل: رسالة الترحيب) |
| `trigger_type` | string | welcome, greeting, goodbye, thank_you, custom |
| `trigger_keywords` | json | الكلمات المفتاحية المُحفزة |
| `reply_text` | text | نص الرد |
| `use_emojis` | boolean | استخدام الإيموجي |
| `tone` | string | friendly, professional, casual |
| `variables` | json | المتغيرات ({customer_name}, {store_name}) |
| `when_to_use` | enum | always, first_message, after_order, custom |
| `priority` | int | الأولوية |
| `is_active` | boolean | نشط |
| `usage_count` | int | عدد مرات الاستخدام |
| `last_used_at` | datetime | آخر استخدام |

#### الردود الافتراضية:
```json
[
  {
    "name": "رسالة الترحيب",
    "trigger_type": "welcome",
    "reply_text": "هلا وغلا! 👋 أهلاً بيك في {store_name}، شنو اقدر أساعدك به اليوم؟ 😊"
  },
  {
    "name": "رسالة الوداع",
    "reply_text": "حياك الله! تشرفنا بخدمتك. إذا احتجت أي شي، احنا موجودين! 🙏"
  },
  {
    "name": "شكراً على الطلب",
    "reply_text": "شكراً لثقتك يا {customer_name}! 🎉 طلبك راح يوصلك قريباً. نتمنى يعجبك! ❤️"
  }
]
```

### 3. جدول `unanswered_questions`

يتتبع الأسئلة التي لم يستطع AI الإجابة عليها.

#### الحقول:
| الحقل | النوع | الوصف |
|------|------|------|
| `id` | bigint | المعرف الفريد |
| `user_id` | bigint | معرف المتجر |
| `conversation_id` | bigint | معرف المحادثة |
| `lead_id` | bigint | معرف العميل |
| `question` | text | السؤال |
| `context` | text | سياق المحادثة |
| `detected_intent` | string | النية المكتشفة |
| `confidence_score` | float | ثقة AI (0-1) |
| `admin_answer` | text | إجابة الأدمن |
| `answered_by` | bigint | الأدمن الذي أجاب |
| `answered_at` | datetime | وقت الإجابة |
| `status` | enum | pending, answered, ignored, added_to_kb |
| `is_reviewed` | boolean | تم المراجعة |
| `occurrence_count` | int | عدد مرات السؤال |
| `category` | string | التصنيف |
| `needs_urgent_attention` | boolean | يحتاج انتباه عاجل |

#### سير العمل:
```
1. AI لا يجد إجابة → يُنشئ سجل في unanswered_questions
2. Admin يُراجع السؤال
3. Admin يُضيف إجابة
4. يُنقل إلى knowledge_base تلقائياً
5. AI يتعلم ويستخدم الإجابة الجديدة
```

## 🎯 الميزات الرئيسية

### 1. البحث الذكي (Smart Search)
```php
// البحث عن أسئلة مشابهة
$similar = AiKnowledgeBase::findSimilar($question, $userId);
```

استخدام MySQL Full-Text Search للعثور على أسئلة مشابهة.

### 2. المتغيرات الديناميكية (Dynamic Variables)
```php
$variables = [
    'customer_name' => 'أحمد',
    'store_name' => 'متجر الملابس'
];

$reply = $fastReply->getFormattedReply($variables);
// Output: "هلا وغلا! 👋 أهلاً بيك أحمد في متجر الملابس..."
```

### 3. التحكم بالإيموجي
```php
$reply->use_emojis = false; // إزالة الإيموجي تلقائياً
```

### 4. الأولويات
- أعلى `priority` = يُستخدم أولاً
- مفيد عند وجود ردود متعددة لنفس الكلمة المفتاحية

### 5. الإحصائيات
- تتبع عدد مرات استخدام كل رد
- معرفة أكثر الأسئلة شيوعاً
- أكثر الردود استخداماً

## 🔄 سير العمل الكامل (Workflow)

### السيناريو 1: عميل يسأل سؤال جديد

```
1. العميل: "هل تقبلون الدفع بالفيزا؟"
   ↓
2. AI يبحث في knowledge_base - لا يجد إجابة
   ↓
3. يُنشئ سجل في unanswered_questions
   ↓
4. يُرسل إشعار للأدمن
   ↓
5. الأدمن يفتح صفحة "أسئلة معلقة"
   ↓
6. يرى السؤال ويضيف إجابة: "نعم نقبل الفيزا وزين كاش وكي كارد"
   ↓
7. يضغط "حفظ وإضافة لقاعدة المعرفة"
   ↓
8. يُنقل السؤال والجواب إلى knowledge_base
   ↓
9. AI يتعلم - المرة القادمة يُجيب تلقائياً ✅
```

### السيناريو 2: تخصيص رسالة الترحيب

```
1. الأدمن يفتح "الردود السريعة"
   ↓
2. يختار "رسالة الترحيب"
   ↓
3. يعدل النص:
   القديم: "هلا وغلا! 👋"
   الجديد: "مرحباً بيك يا زين! 🌟"
   ↓
4. يعطل الإيموجي ❌
   ↓
5. يحفظ
   ↓
6. من الآن - كل عميل جديد يستلم الرسالة الجديدة ✅
```
